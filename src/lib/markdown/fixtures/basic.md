# Semiconductor Lithography

Photolithography transfers a pattern from a **photomask** onto a *light-sensitive* resist. The resolution limit depends on the `wavelength` and the numerical aperture.

## Key ideas

Deep ultraviolet sources use ~~mercury lamps~~ excimer lasers. See [Photoresist](Chemistry/Photoresist.md) for the chemistry, or visit <https://example.com/euv> for background.

### Process window

A ***narrow*** process window makes production yield sensitive to dose and focus variation.
